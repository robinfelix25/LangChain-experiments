# 🚀 LangChain New Architecture Playground (Ollama Focus)

This repository is a lightweight playground to experiment with the **new LangChain architecture**, primarily using **free and local LLMs via Ollama**.

---

## 📌 Goal

* Explore latest LangChain updates
* Build with **local / free LLMs (Ollama)**
* Stay updated with modern LLM workflows

---

## ⚙️ Setup

```bash
git clone <repo-url>
cd <repo-name>
pip install -r requirements.txt
```

Make sure you have Ollama installed and running.
